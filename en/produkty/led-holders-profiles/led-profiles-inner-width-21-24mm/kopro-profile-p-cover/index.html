Basic info

KOPRO profile is designed for universal, interior lighting applications. Beam angle - 45 dgr.

Technical parameters

Material anodized aluminium
Length 1m, 2m, can be cutted on customer's order
Cover milky

Technical drawing

Zastosowanie

  • the KOPRO extrusion is designed to build universal interior lighting fixtures,
  • this profile directs the light beam at an angle of 45° so you can achieve the effect of mild accent illumination and avoid glare,
  • profile can be equipped with two LED strips, side-by-side,
  • KOPRO can be mounted in corners (e.g. bodies of furniture, cabinets, where ceiling meets wall, shop windows),
  • the KOPRO extrusion can be mounted with screws or bolts (suitable for a particular substrate) screwed into the walls of the profile which are not intended for LEDs. Holes for the bolts or screws should be drilled in places designated by the lines inside the extrusion.
  • in drywall ceilings, the profile can be fixed using GP mounting springs, or with use of adhesives.
